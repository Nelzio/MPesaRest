M-Pesa for all

O M-Pesa FA ou simplesmente M-Pesa for all é criado pensando em desenvolvedores de diferentes plataformas, onde não é necessário muito code ou sdk no seu programa para fazer pagamentos via M-Pesa.

O M-Pesa FA é na verdade um SDK online.

Neste momento, o M-Pesa FA está em desenvolvimento, onde se espera muito feedback dos developers.

O M-Pesa FA é desenvolvida e mantida por Nélzio Sitoe.
