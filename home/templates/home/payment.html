Descrição

A chamada da API do C2B é usada como uma transação padrão de cliente para empresa.
Os fundos da carteira móvel do cliente serão deduzidos e transferidos para a carteira móvel da empresa. Para autenticar e autorizar esta transação, o M-Pesa enviará uma mensagem USSD ao cliente para coletar o seu número PIN da móvel. Esse número não é armazenado e é usado apenas para autorizar a transação.

M-Pesa FA

Para usar a API C2B do m-pesa através do M-Pesa FA você tem de fazer uma requisição post para:
https://mpesa-fa.herokuapp.com/api/purchase/
